## Docker Auto Script

### With Binary File
```sh
1. Download docas.zip
2. Extract docas.zip
3. sudo ./docas -u user@127.0.0.1 -p password 
```
### Shell Script
```sh
1. wget https://raw.githubusercontent.com/EPX-PANCA/docker-auto-script/main/setup.sh
2. sudo ./setup.sh -u user@127.0.0.1 -p password 
```
